##  Featured Projects

###  LLM-Guided Vision-and-Language Navigation
> *Research · Wilfrid Laurier University*

An end-to-end VLN agent that navigates indoor Matterport3D environments using natural language instructions. Combines a GPT-4-turbo high-level planner with BLIP captioning, YOLOv8 object detection, MiDaS depth estimation, a PPO RL executor, Spatial Episodic Memory, and a 6-layer guardrail system.

`GPT-4` `BLIP` `YOLOv8` `MiDaS` `PPO` `SLAM` `Matterport3D` `R2R`

---

###  ML-Powered Academic Research Retrieval System
> *Wilfrid Laurier University*

RAG-based research retrieval system combining SciBERT embeddings and Elasticsearch to search, rank, and summarise academic papers. Embedding-based search improves retrieval precision over keyword matching.

`RAG` `SciBERT` `Elasticsearch` `Transformers` `PyTorch` `TensorFlow`

---

###  Real-Time Sports Analytics Pipeline
> *Wilfrid Laurier University*

Real-time player tracking system using YOLOv8 + DeepSORT over match footage. Extracts movement metrics (speed, acceleration, zone occupancy) and forecasts player positioning using LSTM models — 93% accuracy on zone prediction.

`YOLOv8` `DeepSORT` `LSTM` `OpenCV` `Computer Vision`

---

###  Anomaly Detection in Firewall Traffic
> *Wilfrid Laurier University*

Anomaly detection pipeline over network traffic logs using Isolation Forest, Gradient Boosting, and Graph Neural Networks to flag suspicious behaviour and monitoring signals.

`PyTorch` `GNN` `Scikit-learn` `Isolation Forest` `Network Analysis`
